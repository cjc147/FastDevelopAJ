# FastDevelopAJ
Android手机APP快速开发的工程，开发语言为Java

## 功能清单
### JNI部分
* .so包存放于 app/src/main/jniLibs 下各自CPU架构的目录内
* .jar包存放于 app/libs 目录下

### 国际化
* 支持简体中文
* 支持英文

## 第三方包
* OkHttp
* Glide
* Logger
* EventBus
* Gson
* Room
* SmartRefreshLayout
* PhotoView

### 动态权限

## 修改为自己的工程
### 工程级别
